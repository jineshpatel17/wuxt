# Colour Variables

Our colour variable names come from http://www.htmlcsscolor.com/hex/
However, each colour should be assigned to a different, agnostic variable.

The markup provided below is purely to demonstrate the visual
representation of these colors.

### Colours | Monotone

```colours-mono.html
<div class="row">
  <div class="swatch">
    <picture class="card__image bg--light-alpha"></picture>
    <div class="swatch__content">
      <p><strong>$cl__light--alpha:</strong> #ffffff</p>
    </div>
  </div>
  <div class="swatch">
    <picture class="card__image bg--light-beta"></picture>
    <div class="swatch__content">
      <p><strong>$cl__light--beta:</strong> #f5f5f5</p>
    </div>
  </div>
</div>

<div class="row">
  <div class="swatch">
    <picture class="card__image bg--dark-alpha"></picture>
    <div class="swatch__content">
      <p><strong>$cl__dark--alpha:</strong> #333a3c</p>
    </div>
  </div>
  <div class="swatch">
    <picture class="card__image bg--dark-beta"></picture>
    <div class="swatch__content">
      <p><strong>$cl__dark--beta:</strong> #262a2b</p>
    </div>
  </div>
</div>
```
